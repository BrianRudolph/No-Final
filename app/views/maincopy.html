Technologies?

Oh yeah...this page is chalk full of them!

First off this page was created using the Play! Framework. Its an open source framework built using the Java Language. Its like Ruby on Rails...except we get to code in Java instead of Ruby!

We're hosted on Heroku, which hosts all our information on the "cloud" , actually we're just using a single Amazon EC2 cluster....but you get the point.

Heroku is also based on a freemium business model! (We're just using the free package...so please don't dos attack us...an apologies for the slowness)

Some of the CSS is taken from, and the scaffolding of the layout is built using the open source project Twitter Bootstrap

All this code is open source , and can be viewed and downloaded on our Github! Group collaboration and networking effects at its finest...

We're spreading the word via all sorts of cool networks, trying to use network effects as efficiently as possible! Plus a bunch of people are contributing to this project...
